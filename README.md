# 🎉 SmartEvent

Aplicativo desenvolvido com Ionic + Vue para visualização de eventos.

## 🚀 Funcionalidades
- 📋 Listagem de eventos
- 🔍 Tela de detalhes
- ⭐ Favoritar eventos
- 💾 Persistência com localStorage

## 🛠️ Tecnologias
- Ionic
- Vue 3
- Vue Router
